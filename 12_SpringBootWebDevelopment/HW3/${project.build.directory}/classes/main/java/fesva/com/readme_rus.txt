Цель задания

Научиться подключать Hibernate для использования в Spring.

Что нужно сделать

Организуйте хранение данных списка дел приложения ToDoList, используя базу данных.

Должны быть реализованы все запросы: создание дела, удаление дела, обновление дела, получение списка, удаление всего списка.



Рекомендации

Если у вас кириллица приходит в виде знаков ? или в виде нечитаемых символов, то вам необходимо указать в строке подключения к базе данных, что данные будут в UTF-8 кодировке.
Добавьте в строку подключения следующие параметры useUnicode и characterEncoding:

spring.datasource.url=jdbc:mysql://127.0.0.1:3306/tasks?useUnicode=true&characterEncoding=utf8

Если при запуске приложения у вас возникает ошибка
 java.lang.NoClassDefFoundError: javax/xml/bind/JAXBException
вам необходимо добавить недостающую зависимость в pom.xml



<dependency>

    <groupId>javax.xml.bind</groupId>

    <artifactId>jaxb-api</artifactId>

    <version>2.3.0</version>

</dependency>



После того как добавите зависимость, перечитайте файл pom.xml используя сочетание клавиш CTRL+SHIFT+O.

Такая ошибка возникает по причине отсутствия по умолчанию библиотеки JAXB в версиях JDK 11 и выше.
